== Sending tokens in the request

The code snippet below illustrates how to send a JWT token in the `Authorization` request header:

[source, groovy]
.Accessing a protected resource using an Authorization request header
----
String accessToken = //... obtain the access token
HttpRequest requestWithAuthorization = HttpRequest.GET('/' ).header(HttpHeaders.AUTHORIZATION, "Bearer $accessToken") 
HttpResponse<String> response = client.toBlocking().exchange(requestWithAuthorization, String)
----

TIP: Check out the http://guides.micronaut.io/micronaut-security-jwt/guide/index.html[Micronaut JWT Authentication guide] for a tutorial on Micronaut's JWT support.
